How to get your Foursquare data

There are two ways to export your Foursquare/Swarm checkin history. Either method produces JSON files that work with Swarm-FM.

Option 1: Privacy data request

The official way. No coding required, but it can take several days to receive your data.

  1. Go to Foursquare's data request page
  2. Follow the instructions to request your data export
  3. You'll receive an email with a download link when ready (up to 7 days)
  4. Extract the zip — upload the checkins*.json files

Option 2: API export via Pinback

Faster and immediate, but requires a Foursquare developer account.

  1. Visit github.com/lokesh/pinback
  2. Follow the setup instructions to create a Foursquare API app and get an OAuth token
  3. Run Pinback to export your checkins as JSON
  4. Upload the resulting file here

Note: Foursquare is sunsetting City Guide, so the privacy request process may change. The API method via Pinback is a reliable alternative.

Privacy

Your data stays in your browser

All processing runs client-side via Pyodide (Python in WebAssembly). Your Last.fm and Foursquare files are never uploaded to any server.

Geocoding via Nominatim

For checkins that don't already have city data, coordinates rounded to ~10 km are sent to OpenStreetMap's Nominatim service to resolve city and country names. This is the only network request that includes any location information. See the OSM Foundation privacy policy.

Pre-built location cache

A geo_seed.json file is loaded to speed up geocoding. It contains only coordinate-to-city mappings (e.g. "40.1,-74.2" → "New York, US") at ~10 km grid resolution, derived from the developer's own checkin data. It contains no venue names, timestamps, or personally identifying information. When this cache already has a match, no Nominatim request is made.

Local storage

Geocoding results are cached in your browser's localStorage so subsequent runs don't repeat lookups. You can clear this anytime via your browser's developer tools.

No analytics or tracking

This site does not use cookies, analytics, or any third-party tracking scripts.
